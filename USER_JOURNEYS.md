# MVP User Journeys

## Journey #1: "От калькулятора к первому изделию"

**Цель пользователя:** "Я хочу понять, сколько ткани мне купить"

### Полный путь

```
🏠 Экран 1: Главная (гость)
    ↓
    Выбирает "Рассчитать расход ткани"
    ↓
📐 Экран 2: Калькулятор (пошаговый)
    ↓
    Заполняет: тип изделия, размер, рост, ширина ткани, наличие рукава
    ↓
    Нажимает "Рассчитать"
    ↓
📊 Экран 3: Результат расчёта
    ↓
    Видит: "Вам понадобится 2.15 м. Рекомендуем купить 2.3 м"
    ↓
    Выбирает дальнейшие действия:
    - 🛒 Составить список покупок
    - 🧵 Посчитать себестоимость
    - 🤖 Спросить помощника
    - 💾 Создать мою мастерскую ← НАЖИМАЕТ
    ↓
🎉 Экран 4: "Создать мою мастерскую"
    ↓
    Видит: "Ваш первый расчёт готов. Создайте мастерскую и он сохранится навсегда"
    ↓
    Нажимает "Создать мастерскую"
    ↓
👤 Экран 5: Регистрация (15 сек)
    ↓
    Выбирает: Google / Apple / VK / Яндекс / Email
    ↓
    Аутентифицируется
    ↓
🏠 Экран 6: Поздравляем
    ↓
    Видит: "Ваша мастерская готова. Хотите создать первое изделие?"
    ↓
    Нажимает "Да"
    ↓
📝 Экран 7: Назвать изделие
    ↓
    Вводит название (например: "Платье для дочери")
    ↓
    Нажимает "Создать"
    ↓
👗 Экран 8: Карточка изделия
    ↓
    Магия! Расчёт уже внутри
    ↓
    Видит:
    - Название
    - Обложка (пусто)
    - Нужная ткань: 2.3 м (из расчёта)
    - 🤖 "Что дальше?"
    - 📷 Добавить фото
    - 📝 Добавить заметку
```

## Экран 2: Калькулятор (Пошаговый)

### Диз��йн

```
═══════════════════════════
    Расход ткани
═══════════════════════════

Шаг 1 из 5

Что будем шить?

▼ Платье

┌─────────────────┐

│ Брюки           │
│ Юбка            │
│ Рубашка         │
│ Футболка        │

```

### Логика

- **Один вопрос на экран**
- По одному вопросу для удобства на мобильном
- После каждого ответа "Далее" переходит к следующему вопросу
- Вопросы:
  1. Что будем шить? (Платье, Брюки, Юбка и т.д.)
  2. Размер? (32-52)
  3. Рост? (150-200)
  4. Ширина ткани? (140, 150, 160 см)
  5. Есть рукав? (Да/Нет)

## API Endpoints для Journey #1

```
POST /api/calculator/fabric-usage
  → Рассчитать расход ткани

POST /api/auth/register
  → Регистрация пользователя

POST /api/workshops
  → Создать мастерскую (автоматически при регистрации)

POST /api/projects
  → Создать проект (изделие)

POST /api/projects/:id/attach-calculation
  → Прикрепить расчёт к проекту
```
